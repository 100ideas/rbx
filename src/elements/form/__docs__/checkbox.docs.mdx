# Checkbox

This is the form checkbox input.
It is intentionally not styled, as Bulma seeks to preserve cross-browser compatibility and user experience.

<Playground>
  <Label>
    <Checkbox /> Remember me
  </Label>
</Playground>

You can add **links** to it.

<Playground>
  <Label>
    <Checkbox /> I agree to the <a href="#">terms and conditions</a>
  </Label>
</Playground>

### Disable

Use the standard `disable` prop to disable the checkbox.

<Playground>
  <Label disabled>
    <Checkbox disabled /> Save my preferences
  </Label>
</Playground>
