# Radio Button

The mutually exclusive **radio buttons** in their native format.

The `<Radio>` component is a simple wrapper around the `<input type="radio">` HTML elements.
It is intentionally not styled, to preserve cross-browser compatibility and the user experience.

Make sure that the linked radio buttons have the **same value** for their `name` prop.

<Playground>
  <Control>
    <Label>
      <Radio name="answer" /> Yes
    </Label>
    <Label>
      <Radio name="answer" /> No
    </Label>
  </Control>
</Playground>

Use the standard `checked` prop to mark a radio button by default.

<Playground>
  <Control>
    <Label>
      <Radio name="foobar" /> Foo
    </Label>
    <Label>
      <Radio name="foobar" defaultChecked /> Bar
    </Label>
  </Control>
</Playground>

### Disable

Use the standard `disable` prop on the `<Radio>`, and the `disabled` prop on the `<Label>` to disable it.

<Playground>
  <Control>
    <Label>
      <Radio name="rsvp" /> Going
    </Label>
    <Label>
      <Radio name="rsvp" /> Not going
    </Label>
    <Label disabled>
      <Radio name="rsvp" disabled /> Maybe
    </Label>
  </Control>
</Playground>
