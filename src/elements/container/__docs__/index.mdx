# Container

## Container

The container centers its content horizontally.

<Playground>
  <Frame updateHeightDelay={0}>
    <Container>
      <Notification>
        This container is <strong>centered</strong> on desktop.
      </Notification>
    </Container>
  </Frame>
</Playground>

### Fluid

Use the `fluid` prop for the container to fit horizontally with padding.

<Playground>
  <Container fluid>
    <Notification>
      This container is <strong>fluid</strong>: it will have a 32px gap on
      either side, on any viewport size.
    </Notification>
  </Container>
</Playground>

### Breakpoint

<Playground>
  <Frame>
    {DEFAULTS.breakpoints.map(breakpoint => (
      <Container breakpoint={breakpoint} key={breakpoint}>
        <Notification>
          This container is <strong>fullwidth</strong> <em>until</em> the
          <code>{breakpoint}</code> breakpoint.
        </Notification>
      </Container>
    ))}
  </Frame>
</Playground>

> Therefore, they will also accept all props for the `as` component.

> These component also accept all props supported by the `Generic` component.
