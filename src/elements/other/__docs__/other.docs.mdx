Often, these components aren't useful on their own.

### Delete

Frequently used with the `<Modal>` and `<Notification>` components.

<Playground>
  <Delete />
</Playground>

Use the `size` prop of `<Delete>` to change its size.

<Playground>
  {() => {
    const sizes = DELETE_DEFAULTS.sizes.map(size => ({ name: size, size }));
    sizes.splice(1, 0, { name: "-", size: undefined });

    return (
      <Table bordered>
        <thead>
          <tr>
            <th><code>Delete</code> size</th>
            <th>Result</th>
          </tr>
        </thead>
        <tbody>
          {
            sizes.map(({size, name}) => (
              <tr key={name}>
                <td><code>{name}</code></td>
                <td><Delete size={size} /></td>
              </tr>
            ))
          }
        </tbody>
      </Table>
    );

}}

</Playground>
