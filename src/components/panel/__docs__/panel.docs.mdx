# Panel

A composable **panel**, for compact controls

<div className="content">
  <p>
    The <code>{"<Panel>"}</code> is container for several types:
  </p>
  <ul>
    <li>
      <code>{"<Panel.Heading>"}</code> as the first child
    </li>
    <li>
      <code>{"<Panel.Tabs>"}</code> for navigation
    </li>
    <li>
      <code>{"<Panel.Block>"}</code> which can contain other elements, like:
      <ul>
        <li>
          <code>{"<Control>"}</code>
        </li>
        <li>
          <code>{"<Input>"}</code>
        </li>
        <li>
          <code>{"<Button>"}</code>
        </li>
        <li>
          <code>{"<Panel.Icon>"}</code>
        </li>
      </ul>
    </li>
  </ul>
  <p>
    The <code>{"<Panel.Block>"}</code> can be an anchor tag <code>{"<a>"}</code>{" "}
    or a label <code>{"<label>"}</code> with a checkbox inside.
  </p>
</div>
