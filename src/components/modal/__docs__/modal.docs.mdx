# Modal

A classic **modal** overlay, in which you can include any content you want.

<Content>
  <p>
    The <code>{"<Modal>"}</code> structure is very simple:
  </p>
  <ul>
    <li>
      <code>{"<Modal>"}</code>: the main container
      <ul>
        <li>
          <code>{"<Modal.Background>"}</code>: a transparent overlay that can
          act as a click target to close the <code>{"<Modal>"}</code>
        </li>
        <li>
          <code>{"<Modal.Content>"}</code>: a horizontally and vertically
          centered container, with a maximum width of 640px, in which you can
          include <em>any</em> content
        </li>
        <li>
          <code>{"<Modal.Close>"}</code>: a simple cross located in the top
          right corner
        </li>
      </ul>
    </li>
  </ul>
</Content>

### Image modal

Modals can contain anything, for example, an image.
